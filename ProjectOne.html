Prey and Predator

Overview

The prey and predator system is depends on the dependence one on another. I have divided the whole system in to four parts and implemented them into as a each individual part. Those are:
1. Movement.
2. Resource consumption.
3. Reproduction.
4. Evolution.

Project and Tools

Project link: click here for the Github project
I used Unity Engine, Visual Studio and GitHub for the development of the project.

Simulation Video

The simulation start with spawing the prey and predator and they move around with the border of the screen. Prey which is in green color, get destroyed and spawned when collided with predator. Predator, which is in red color, has a radius where it will chases the prey with in radius. This simulation is endless when it is played. Predator has dialouge text where it give state of it behaviour.


Responsibilities

I designed code and logic for the simulation, and built systems around prey and predator.
I built systems for simulation where everything changes according to the behaviour other system
I developed dialouge system for the predator.

Code design

I have divided code in to components which are implemented by OOPS and Game Programming Patterns.
For instance,
Movement: MoveTowards, Core is a target position.
Reproduction: Coroutine, Inheritance for different objects.
Radius: Line Renderer MathF.Sin and Cos (Degrees to radians)
Collision: Physics.OverLap( ) range Follow Target
Destroy: List , Collider hit which reset the object position to start location
Evolution: State Design Pattern, to define states which are move, evolve, old and reproduce.
